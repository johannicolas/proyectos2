la aplicacion creada es una aplicacion maven. se utilizo el IDE Netbeans. las pruebas realizadas se hicieron en el servidor glassfish con BD Oracle. la aplicacion
se construyo con los estandares de Java (JSF,EJB,JPA).


para ejecutar la aplicacion es necesario 

1) crear un data source en el servidor. el archivo de persistencia se encuentra en la ruta:
lamejorcocina\src\main\resources\META-INF\persistence.xml 

2) ejecutar los scripts del archivo bd en la base de datos.

3) desplegar el archivo lamejorcocina-1.0-SNAPSHOT.war en el servidor.
